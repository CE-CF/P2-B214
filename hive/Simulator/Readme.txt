If no controller connected: Move the Drone with the arrow keys and move the second drone with ikjl
If a controller is connected: Move on drone with wasd, on drone with left analog stick, and one with the controller d-pad arrows.
If Tello mode on: Use the communication tester to send commands to the two drones that have IP 0.0.0.0 and 0.0.0.1
No controller: Move the Camera with WSAD
Controller: Move the Camera with the right analog stick
Exit with escape or the X on the top right
In the start menu you can choose 4 different resolutions, and if you want full screen or not

How to send commands:
  The commands are of format IP Address: Command
  Example: 0.0.0.0: takeoff
